Total of 3 clusters, each cluster has 1 control plane, 1 worker node
Specs of each VM/nodes - 2 CPU, 4GB of memory, 20GB of disk space
Hypervisor - Proxmox VE 9
OS - Debian 12

Cluster Names:
- central-hub   (hosts ArgoCD - manages prod-cluster and dev-cluster)
- prod-cluster
- dev-cluster

ArgoCD Setup:
- Version v3.3.6
- Hub-Spoke model: ArgoCD runs on central-hub
- central-hub's ArgoCD manages deployments to prod-cluster and dev-cluster
- Remote clusters registered via: argocd cluster add
- GitOps source: GitHub repository (One Mono-Repo for all clusters)
